# GitHub Repository Setup

## Create Repository on GitHub

1. Go to: https://github.com/new
2. Repository name: **smartchef**
3. Description: "A Flutter recipe app with AI-powered recipe generation"
4. Choose Public or Private
5. **DO NOT** check any initialization options (README, .gitignore, license)
6. Click "Create repository"

## Push Your Code

After creating the repository, run:

```bash
cd /Users/nityam/development/Flutter\ Projects/smartchef
git remote set-url origin https://github.com/SharmaNityam/smartchef.git
git push -u origin main
```

If you prefer SSH:
```bash
git remote set-url origin git@github.com:SharmaNityam/smartchef.git
git push -u origin main
```

## Verification

After pushing, verify no secrets are in the repository:
- ✅ .env file is in .gitignore
- ✅ Firebase config files are in .gitignore  
- ✅ Only .example template files are committed
- ✅ All API keys are loaded from environment variables

Your code is ready to push once the repository is created!
